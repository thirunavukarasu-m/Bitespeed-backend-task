Identify Contact API

This endpoint is used to identify and manage user contacts based on phone number and/or email.

Endpoint: POST /identify

Base URL (Production): https://bitespeed-backend-task-gvq5.onrender.com/identify

Request Payload

Send a JSON body like this:

{ "phoneNumber": "9876543210", "email": "user@example.com" }

How to Test

You cannot test this via browser directly. Please use API tools like:

  • Hoppscotch
  • Postman

Choose POST method, enter the endpoint, and send a raw JSON body as shown above.

Once submitted, you will receive a JSON response with the resolved primary and secondary contact details.
